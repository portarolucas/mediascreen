FAQ

Voici une liste de question/réponse qui pourront être utile pour une bonne utilisation de l’application MediaScreen.

1. QUE FAIRE SI LES ÉCRANS DE DÉFILENT PLUS ?

Plusieurs sources de problèmes peuvent causer un disfonctionnement de la lecture des écrans dans une séquence :

  • La connexion : si le périphérique n’est plus connecté au serveur Web il finira le défilement de sa séquence et s’arrêtera (actualisation des écrans à chaque fin de séquence)
  • La séquence n’est plus disponible sur le serveur : cela entraînera un arrêt imprévu du défilement
  • Votre token n’est plus disponible (suppression par l’administrateur)

Des messages d’aides/explicatifs seront alors diffusé sur les écrans.

2. COMMENT TESTER L’APPLICATION

Pour tester l’application veuillez vous rendre ici puis utiliser les logs (NOM USER: ADMIN | PASSWORD : ADMIN) pour vous connecter. Vous pourrez ensuite créer des séquences et écrans, vous pourrez ensuite visualiser les séquences créé via un lien générer par l’application.
